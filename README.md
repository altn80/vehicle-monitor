# vehicle-monitor 

 http://localhost:4200/

## Build/Run Requirements
* apache maven 3
* jdk 1.8
* node 10.15.2
* angular cli 8.3.9
* docker 18.09.7
* docker-compose 1.21.0

## Build Instructions (vehicle-monitor-backend)

  ```
  mvn clean package
  ```

## Build Instructions (vehicle-monitor-frontend)

  ```
  npm install
  npm run-script build
  ```

## Docker Instructions

  ```
  docker-compose up
  ```
